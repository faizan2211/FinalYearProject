# FinalYearProject
Geo-Assistant (AI Smart Travel Recommendation System)
Geo-Assistant is an AI-powered, location-aware travel application that automatically detects when users enter a new city and provides intelligent, unbiased recommendations using GPS geofencing and aggregated reviews from multiple sources. Built with Flutter, Python, MongoDB, Google Maps API, and an AI chatbot, it enhances travel experiences through personalized assistance, navigation, and rich destination information.
